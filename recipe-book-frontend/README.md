# Recipe Book Frontend

Фронтенд для бэкенда «Книга рецептов».

## Запуск в Visual Studio Code

```bash
npm install
npm run dev
```

Бэкенд должен быть запущен на `http://localhost:8080`.

Если нужен другой адрес сервера, создай файл `.env`:

```bash
VITE_API_BASE_URL=http://localhost:8080/api
```

## Структура

```text
src/app
src/pages
src/features
src/entities
src/shared
```
